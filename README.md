# 🎒 寒假作业大作战：把作业变成打怪升级！


大家好，我是一个被孩子漏写作业搞奔溃的妈妈。
作为一个二年级小学生的妈妈，每到寒假，管理孩子的作业进度总是让人头大。为了缓解孩子的“作业焦虑”，也为了让我（家长）能更直观地掌握进度，我写了一个小网页，把作业变成了“通关游戏”。

### 🎮 它能做什么？
1.  **每日任务清单**：每天要做什么，一目了然，做完一项打个钩，屏幕会撒花庆祝 🎉！
2.  **战况总览**：像游戏血条一样显示进度，看着进度条一点点变满，孩子特别有成就感。
3.  **防误触机制**：内置家长 PIN 码锁，防止孩子误删或随意修改任务总量。
4.  **完全免费 & 安全**：不需要注册账号，不需要下载 App，数据都存在你自己的手机里。



### 📱 怎么用？
1.  点开链接：[https://homework-4p4.pages.dev/]
2.  **推荐操作**：在微信里点开后，选择“在浏览器打开”，然后选择“添加到主屏幕”（iphone和安卓手机添加图标的方式不太一样，建议你们可以去问问豆包！）。这样桌面上就会出现一个小书包图标 🎒，以后点图标就能用啦！

希望这个小工具能帮各位家长“解放双手”，祝孩子们寒假快乐，作业顺利通关！🚀
